## Reflection

### What I Learned
- Workflow **Plan-First** memaksa saya berpikir sebelum coding, hasilnya lebih terstruktur
- **RTCC-O** framework membantu menggunakan AI secara efektif — bukan copy-paste, tapi sebagai tool
- Meskipun saya sudah terbiasa dengan Next.js + TypeScript, kembali ke plain HTML/CSS mengingatkan saya pentingnya fundamentals
- CSS Grid + CSS Variables sangat powerful bahkan tanpa framework

### Key Takeaway
> AI adalah alat, bukan pengganti pemahaman. Setiap baris kode yang saya submit, saya bisa jelaskan kenapa ada di situ.

### AI Usage Disclosure
Digunakan Claude AI sebagai coding assistant untuk:
- Membantu struktur awal HTML
- Menyarankan CSS patterns
- Dokumentasi plan/ files

Semua output direview, dipahami, dan dimodifikasi sesuai kebutuhan saya. Tidak ada raw AI output yang disubmit tanpa pemahaman.
