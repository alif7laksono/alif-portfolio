# 01 — Brainstorm

## Project Goal
Build a personal portfolio website that showcases my identity as a Frontend Developer, highlights real production projects, and serves as a professional online presence for recruiters and clients.

## Who Is This For?
- Recruiters looking for frontend developers
- Potential freelance clients
- Bootcamp evaluators (KAreerr Elevation Vol. 2)

## What Must It Communicate?
1. I am a real frontend developer — not just a learner
2. I have live, production-ready projects anyone can visit
3. My tech stack: Next.js, TypeScript, Tailwind CSS, MongoDB, AWS S3
4. I am professional, detail-oriented, and ship working products

## Design Inspiration
- Reference: passes.com — dark theme, bold typography, teal/cyan accent color
- Style: Modern, minimal-dark, glassmorphism hints, high-contrast

## Content Sections (Brainstormed)
- Hero — bold statement + CTA
- About — who I am + tech stack tags
- Skills — tech categories with icons
- Projects — 4 real live projects
- Experience — work history timeline
- Contact / Footer — email + GitHub + LinkedIn

## Key Principle Applied
> "Raw AI output you don't understand = AUTO-FAIL"

Every line of HTML, CSS, and JS in this project I can fully explain. I wrote and reviewed all code personally, using AI only as a coding assistant — not as a replacement for understanding.

## AI Prompt Used (RTCC-O)
**Role:** You are a senior frontend developer and UI/UX designer  
**Task:** Help me brainstorm and plan a personal portfolio website  
**Context:** I am Alif Wiji Laksono, a frontend developer from Semarang. I use Next.js, TypeScript, Tailwind CSS, MongoDB, and AWS S3 in real projects  
**Constraints:** Must use only HTML, CSS, and vanilla JS (no frameworks for this homework). Must be responsive, semantic, and explainable  
**Output:** A structured brainstorm document with sections, design direction, and content plan
